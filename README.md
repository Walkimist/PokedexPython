# Pokédex Python app
This is a project that aims to display a pokedex with all pokémon, moves, abilities and types using requests from PokéAPI

## Current Features:
- Search pokémon by id or name, displaying:
  * Name and national Dex Number
  * Genus
  * Height and weight
  * Abilities
  * EV Yield, Catch rate, Base friendship, Base exp., Growth rate
  * Egg groups, Egg cycles
  * Gender ratio
  * Individual stats and base stat total
  * Defensive type relationships
  * Baby, Legendary or Mythical classification
  * Color, Shape and if there are gender differences
  * Moves learned by:
    * Level
    * TM/Tutor
    * Egg moves

## Planned Features:
- Search abilities by name or id
- Search moves by name or id
- Search types by name or id
- Display evolution line data in pokémon search
- Window UI
